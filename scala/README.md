## Set up 

### Install CmdStan  
Steps : https://github.com/stan-dev/cmdstan/wiki/Getting-Started-with-CmdStan  

Clone CmdStan
```commandline
git clone https://github.com/stan-dev/cmdstan.git --recursive
```

Build all tools
```commandline
cmdstan/make build
```

Set CmdStanHome
```commandline
export CMDSTAN_HOME=~/cmdstan/
```
